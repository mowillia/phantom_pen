Usage Tips

Suggestions for How to Optimize Performance
  • It's sometimes better to leave off the period: If you leave off the final punctuation in your entry, the program will finish your sentence for you and will be more likely to continue with the same topic in the following sentences.
  • Start with the 117M model: The 117M model is quicker than the 345M model and often leads to good enough results. Only move to the larger model if you want more coherent results than the 117M output
  • GPU Availability: On the home page check whether the GPU is available. If the GPU is not available, you should set the Length to smaller values and use the 117M model in order to have reasonable generation times.
  • Keep Creativity less than 1: Although you can set Creativity to be greater than 1, doing so leads to less coherent output.
  • The Output is Probabilistic: For high creativity (i.e., creativity close to 1), the output of the generator is never the same given a specific prompt. So if the ouput is less than coherent, you might have gotten unlucky and just need to run the generator again.
  • Start with Shorter Lengths: As a first trial, you should use a shorter length (i.e., the default of '100') to see if you like the output. The longer the length you set, the more time the program takes to produce text.
  • Choose your class carefully: For the "simple generate" function, the Class you choose determines the corpus the program selects as a basis for style and content. To maximize the probability that your output is topically relevant, think about what class your entered text best corresponds to.
  • Style often begets style: Although the program reproduces the style found in the corpus, it also searches the corpus for style that mateches the style (sentence structure and diction) of the entered text.
  • Shorter entered text is better: It is easier for the program to match the content of entered text when that text is short (i.e., 3-5 sentences). Longer text is often seen as an independent essay after which the program might start writing an entirely new essay.
  • Use "Classify, Extract, and Generate" for longer entered text: For longer texts (i.e., a few a paragraphs), use the "classify, extract, and generate" function to extract the few key sentences that could be used as a prompt for the program.
